Want to see your name here? Contribute to Abby!

1. **Code Contributions**: Submit a pull request that gets merged
2. **Documentation**: Improve or add documentation
3. **Bug Reports**: Report detailed, reproducible bugs
4. **Community Support**: Help other users in Discord
5. **Testing**: Participate in beta testing new features
