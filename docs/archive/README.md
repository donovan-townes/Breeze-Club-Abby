## Why Archive?

Documents are archived (not deleted) when they:
- Have been superseded by newer, more comprehensive documentation
- Document point-in-time fixes that are no longer relevant
- Provide historical context for architectural decisions

## Maintenance Policy

Archived documents are:
- **Read-only** — Not updated for current changes
- **Referenced** — May be linked from current docs for historical context
- **Preserved** — Kept for long-term institutional memory

## When to Consult Archive

- Debugging legacy issues
- Understanding historical architectural decisions
- Researching evolution of specific systems
- Auditing past changes
